# Online Store API


## Build the project
    $ mvn clean install

## Run the project

    java -jar target/online-store-api-1.0.0-SNAPSHOT.jar 
    
## Default Path

http://localhost:8081


Add Product
http://localhost:8081/product/add

List Product
http://localhost:8081/product/list

Create Order
http://localhost:8081/order/create

List Order
http://localhost:8081/order/list
